## How to get started with this oriject

First, clone this repository. Move into it on the command line, then:

```bash
> bundle install
> bin/rails db:create
> bin/rails db:migrate
 ```
 
 To run the tests:
 
 ```
> bundle exec rspec # Run the tests to ensure it works
```
To start the server:

```
rails s
```
Then visit localhost:3000.
