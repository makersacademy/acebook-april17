## User stories


```

As a user
so that i can register to use a social media platform
I want to be able to sign up for the service


As a user
so that i can sign in to the social media platform
I want to be able to log in to the service


As a user
so that i can see who is saying what
I want to be able to see everyones social media posts on one page (most recent first)


As a user
so that i can let people know what im thinking
I want to be able to add a post to the website


As a user
so that i can remove one of my posts
I want to be able to delete one of my previous posts


As a user
so that i can make a change to one of my posts
I want to be able to edit one of my previous posts


As a user
so that i can interact with other users
I want to be able to indicate that i like someone elses post


As a user
so that i can give feedback to abother user
I want to be able to add a comment to another users post

```
